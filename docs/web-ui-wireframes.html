Getting Started

Learn how to use GetReceipts to extract knowledge from content

What is GetReceipts?

GetReceipts helps you extract, organize, and share knowledge from podcasts, videos, and documents. The system automatically:

  • Downloads and transcribes audio/video content
  • Extracts claims, insights, and key information using AI
  • Organizes everything into a searchable knowledge base
  • Uploads to GetReceipts.org for web-based review and sharing

The local daemon runs on your computer to handle processing, while the web interface provides a seamless experience for submitting content and reviewing results.

‚úÖ Quick Start Guide
Get up and running in 3 simple steps
1

Configure Your AI Provider

Go to Settings and add your API key. You can use:

  • OpenAI - GPT-4o for high-quality extraction
  • Anthropic - Claude for excellent reasoning
2

Process Your First Content

Go to the main Contribute page and:

  • Paste a YouTube URL (or multiple URLs)
  • Ensure "Full Pipeline" and "Auto-upload" are enabled
  • Click "Process" and wait for completion
3

Review on the Web

Once processing completes, your claims are uploaded to GetReceipts.org. Visit your Dashboard to:

  • Search and filter extracted claims
  • Edit or remove inaccurate claims
  • Organize claims into collections
  • Share insights with others

üìä Feature Distribution Summary

‚úÖ Confirmation: Standard Processing Features

Yes, "standard YouTube processing" and "standard audio processing" refer to the existing desktop app features that have been fully re-implemented in the web UI.

  • Standard YouTube Processing: User pastes YouTube URL ‚Üí Downloads audio ‚Üí Transcribes with Whisper ‚Üí Extracts claims with LLM ‚Üí Uploads to GetReceipts
  • Standard Audio Processing: User drops local audio/video file ‚Üí Transcribes with Whisper ‚Üí Extracts claims with LLM ‚Üí Uploads to GetReceipts

The web UI reuses 100% of the backend processing code from the desktop app via the FastAPI daemon. The only difference is the UI layer (web forms instead of PyQt6 GUI).
